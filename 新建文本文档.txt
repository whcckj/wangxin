使用方法

安装Node.js环境：https://nodejs.org/

安装git: https://git-scm.com/

执行以下命令：

git clone https://github.com/sitexa/anires.git
cd anires
npm install
npm run dev
部署方法

编辑 config/index.js，修改第 10 行的 assetsPublicPath，值为 项目名/public。如果你没有修改项目名 anires，则可跳过此步骤。

编译、上传

bash
npm run build
git add .
git commit -m "update"
git push
开启 GitHub Pages 功能, 或者，把生成的目标文件public/*放在你的web服务器上。
